# Templates y variables requeridas

Esta sección describe las variables mínimas que deben proveerse al renderizar las plantillas incorporadas de Genesis Engine.

## Backend FastAPI
- `project_name`
- `description`
- `version`
- `entities`
- `database_type`

## Backend NestJS
- `project_name`
- `description`
- `port`
- `entities`

## Frontend Next.js
- `project_name`
- `description`
- `typescript`
- `styling`
- `state_management`

## SaaS Basic
- `project_name`
- `description`
